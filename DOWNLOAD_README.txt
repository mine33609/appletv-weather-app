# 🍎 Apple TV Weather App - Download Package

このZIPファイルには、Apple TV対応の気温予測アプリの全ファイルが含まれています。

```
appletv-weather-app/
├── index.html          # メインHTMLファイル（8.4KB）
├── css/
│   └── style.css      # スタイルシート（12.6KB）
├── js/
│   └── main.js        # JavaScript（21.8KB）
```

---

## 🚀 クイックスタート

### **方法1: ローカルでテスト**
1. ZIPファイルを解凍
2. `index.html`をブラウザで開く
3. 動作確認

### **方法2: Netlifyにデプロイ**
1. ZIPファイルを解凍
2. [Netlify](https://www.netlify.com/)にアクセス
3. フォルダをドラッグ&ドロップ
4. 公開URLを取得
5. Apple TVのSafariで開く

### **方法3: GitHubに公開**
1. ZIPファイルを解凍
2. [GitHub](https://github.com/)で新規リポジトリ作成
3. ファイルをアップロード
4. Netlifyと連携してデプロイ

---

## 📱 Apple TVでの使い方

1. **Safari** アプリを起動
2. 公開URLを入力
3. リモコンで操作：
   - **←→**: 要素間の移動
   - **タッチパッド**: 選択
   - **↑↓**: ドロップダウン内の移動

---

## ✨ 主な機能

- 🌡️ **超大型気温表示**: 3メートル離れても視認可能
- 🎮 **Apple TV Remote対応**: 完全キーボード操作
- 📊 **時間別予測グラフ**: 過去24h + 未来24h/48h
- 🌧️ **降水量表示**: 気温と同時表示
- 🔄 **自動更新**: 5分ごとに最新データを取得
- 🏙️ **日本主要10都市対応**: 東京、大阪、名古屋など
